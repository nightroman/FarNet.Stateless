TOPIC
    about_FarNet.Stateless

SHORT DESCRIPTION
    Interactive workflows using dotnet-state-machine/stateless.

LONG DESCRIPTION
    The module provides helpers for the "Stateless" library.

MODULE COMMANDS
    Invoke-Stateless
        Starts or resumes the Stateless script or checkpoint.

    Invoke-StateMachine
        Invokes the state machine interactive loop.

    Show-StateMachine
        Shows the state machine graph in a generated HTML page.
